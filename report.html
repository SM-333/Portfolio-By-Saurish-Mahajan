    • WEBSITE REPORT

    In this report,I will outline the process and key elements involved in creating a professional portfolio website for showcasing my designing skills and achievements in the field of web development. This portfolio showcases my experience, education, skills and creativity.

    1.PLANNING

    The first phase of the project involved planning and some necessary research was executed. I started by identifying the key elements of my profile.

    2.UI AND UX

    These both are the most crucial stages of any website as it tells hows the website gonna look and how it will work. My aim was to create a visually appealing and easy to use interface. Some of the key considerations while making this website were: To create an asthetic design that is visually appealing, To create a responsive layout, Maintaining the color scheme and typography for branding.

    3. DEVELOPMENT

    The development phase involved implementing the design and functionality of the website. I chose to use HTML, CSS, and JavaScript for the front-end development,enhancing interactivity by adding animations and a crazy background.

    4. TESTING

    To ensure the quality of the website i did validation on validator and here are some screenshots regarding that at the end

    The development of this portfolio website has been a valuable learning experience, allowing me to apply my knowledge and skills in web development to create a professional and engaging platform. Moving forward, I plan to continue updating and improving the site to ensure that it remains a relevant and accurate representation of my skills and experiences. Thank you for the opportunity to complete this Web Development Assignment and create a portfolio website that reflects my passion and dedication to the field of web development.
